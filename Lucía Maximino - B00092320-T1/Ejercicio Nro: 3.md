# Ejercicio Nro: 3

## Enunciado
¿Qué ventajas y desventajas tienen los diagramas UML?

## Resolución
Los diagramas UML tienen como principal ventaja que permiten representar de forma visual la estructura y el comportamiento de un sistema, facilitando la comprensión tanto para desarrolladores como para otros miembros del equipo. Ayudan a organizar las ideas, detectar errores de diseño antes de programar y mejorar la comunicación.
Sin embargo, también tienen desventajas. En algunos casos pueden volverse complejos y difíciles de mantener si el sistema cambia mucho. Además, requieren tiempo para elaborarlos y, si no se actualizan, pueden quedar desactualizados respecto al sistema real. En metodologías ágiles, a veces se prioriza más el desarrollo funcional que la documentación extensa.
