# Ejercicio Nro: 2

## Enunciado
¿Qué es para vosotros el desarrollo de software?
    -¿Ejemplos de desarrollo de software en los que hayas participado? ¿Ejemplos que conozcas?
    -¿Cuál es el papel del desarrollador en el desarrollo de un proyecto software?
    -¿Hay otros papeles importantes en el desarrollo, aparte del desarrollador?
    -¿Cuándo se ha desarrollado un buen software?

## Resolución  
El desarrollo de software es un proceso en el que se diseñan, construyen y mejoran sistemas o aplicaciones para resolver problemas concretos o cubrir necesidades de los usuarios.
El papel del desarrollador es importante, ya que se encarga de transformar los requerimientos en una solución funcional, lo cual incluye programar, realizar pruebas, corregir errores y colaborar con el equipo para lograr un producto de calidad.
Además del desarrollador, existen otros roles importantes, como el analista, que se encarga de entender las necesidades del cliente; el tester, que verifica que el sistema funcione correctamente; y el project manager o scrum master, que organiza y coordina el trabajo del equipo.
Un software se considera bueno cuando cumple con ciertas métricas. Por un lado, que sea funcional, es decir, que realmente resuelva el problema para el cual fue creado. Por otro lado, que sea robusto, lo que implica que esté bien construido, funcione correctamente y sea capaz de responder de forma adecuada ante distintos escenarios. Además, es importante que sea fácil de usar y de mantener, que aporte valor al usuario y que pueda adaptarse a cambios con el tiempo.
